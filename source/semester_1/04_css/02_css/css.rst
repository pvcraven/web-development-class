How to specify colors
^^^^^^^^^^^^^^^^^^^^^

Follow this W3Schools tutorial on how to specify colors:

http://www.w3schools.com/colors/default.asp

**Additionally:** You can also do three digit colors.
``#FFF`` is the same as ``#FFFFFF``.
